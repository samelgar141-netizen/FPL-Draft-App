# fpl draft

The data collected in the Jupyter Notebook attatched is used to build the following [tableau dashboard](https://public.tableau.com/views/FPLDraftLeague/DraftRoom?:language=en-GB&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

To find your draft league id, use the following link, there is a section at the end which contains all your leagues, ensure you are logged into FPL.

https://draft.premierleague.com/api/bootstrap-dynamic

Update the config.json file with your draft league id and the folder where you'd like to store the output files.

If you would like to recreate the dashboard for your own draft league then you can then download the tableau dashboard and replace the data source with the output data files for your draft league to see it update!
